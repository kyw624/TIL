# 정수 데이터 리스트 한 줄에 출력하기

## **print end 옵션**

```python
a = [1, 2, 3]

print(a[0], end=' ')
print(a[1], end=' ')
print(a[2])
# 1 2 3
```

<br />

## **반복문**

```python
a = [1, 2, 3]

for i in a:
    print(i, end=' ')
# 1 2 3
```

<br />

## **내장함수 join**

```python
a = [1, 2, 3]

print(" ".join(map(str, a)))
# 1 2 3
```
